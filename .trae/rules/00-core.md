# Kilo Core Rules

- Treat `AGENTS.md` as the repo baseline.
- For small fixes, do not enter architecture-style planning.
- Do not recursively crawl large directories unless the task explicitly asks for an audit.
- If the task touches 1-3 files and has no architecture impact, search narrowly, patch minimally, validate, stop.
- If the user asked a question only, answer it without editing.
- If the requested behavior already exists, say so without making noise.
